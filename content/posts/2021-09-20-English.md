# Practice ICES

* Issue
  - state the issue
  - I'm going to talk about
* Cause
  - state why
  - This is due to the fact,...
  - Objective style
* Effect
  - state the reason or result from the cause
  - Because, ...
  - As a result, ...
* Solution
  - state the solution
  - To solve this, ...
